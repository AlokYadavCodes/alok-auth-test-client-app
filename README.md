# OIDC Identity Demo

This is a small test client app for [Alok Auth](https://alok-auth.vercel.app/).

It lets you sign in with Alok Auth and then shows a simple profile page.

You can also use it with other OAuth or OIDC providers. You just need to change the env values.

The Alok Auth repo is here: [AlokYadavCodes/alok-auth](https://github.com/AlokYadavCodes/alok-auth).

## Setup

1. Clone the repo:

```bash
git clone https://github.com/AlokYadavCodes/alok-auth-test-client-app.git
cd alok-auth-test-client-app
```

2. Install the packages:

```bash
npm install
```

3. Make a `.env` file from `.env.example`.

## Run

```bash
npm run dev
```
